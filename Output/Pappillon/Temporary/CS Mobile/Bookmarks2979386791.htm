Bookmarks

Creating a Bookmark in CenterScape Mobile is a simple function that allows you to bookmark any screen in the CenterScape Mobile so that you can come back to it later.

Bookmarks created and used in CenterScape Mobile do not affect and do not interact with Bookmarks in the CenterScape main web console – they are two separate sets of bookmarks.

Creating a Bookmark

Once at a place you want to Bookmark, select the Main Menu button, highlighted below:

Then select the Bookmarks… button from the Main Menu:

A Bookmarks dialog box will appear containing all of your stored Bookmarks like the one below. This box will be blank upon first use. This menu will allow you to navigate to your stored Bookmarks in CenterScape Mobile.

To Create a Bookmark, select the Add Bookmark button (highlighted in the image below) at the top right corner of the Bookmarks dialog box:

Then the New Bookmark box will appear similar to as shown in the below screen, and you can then input your Bookmark name, and select the OK button to save your new Bookmark.

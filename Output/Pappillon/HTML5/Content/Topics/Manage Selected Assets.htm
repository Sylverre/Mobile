Manage Multiple Selected Assets

Select multiple assets from the assets list to perform an action on all selected assets at once. Assets can be exported, edited, or deleted in groups, and reports or graphs viewed for asset attributes that have recorded values.

  1. Navigate to User Console > Assets and select an asset link. Use the filters if necessary to restrict the number of assets appearing in the asset list.
  2. Select the assets to be managed. Use ctrl-click to highlight multiple separate assets, or shift-click to highlight a sequential group of assets.
  3. Right-click to access available actions to be performed on the selected assets.

 

Some action items can be performed on all assets currently listed, as well as those that have been highlighted. Choose Selected or All.

Edit An Attribute for Multiple Assets

  1. Navigate to User Console > Assets and select an asset link. Use the filters if necessary to restrict the number of assets appearing in the asset list.
  2. Select the assets to be edited. Use ctrl-click to highlight multiple separate assets, or shift-click to highlight a sequential group of assets.
  3. Within the selected range, click the column of the attribute to be edited.

Only those attributes currently in the columnar display can be edited in group. Available attributes are determined by View. Click the right edge of a column header to access the list of available columns.

 

  1. Right-click for menu, and select to edit the attribute for Selected or All.
  2. Click OK.

Attribute Reports/ Graph

Attributes for selected assets can be viewed as tabular data or in graph form if value changes have been recorded for that attribute. Right-click in an attribute column to view menu options including report/graph if available.

  1. Navigate to User Console > Assets and select an asset link. Use the filters if necessary to restrict the number of assets appearing in the asset list.
  2. Select the assets to be included in the report or graph. Use ctrl-click to highlight multiple separate assets, or shift-click to highlight a sequential group of assets.
  3. Within the selected range, right-click in the column of the attribute to be reported.

An Attribute must have the Record Value Changes option selected in order for reports or graphs to be enabled in the Assets list.

  1. A report or Report/Graph pop-up window appears. You can change the controls and display, Run, or Export the report or graph from this window.  Click Close when finished.

Export Selected Assets

  1. Navigate to User Console > Assets and select an asset link. Use the filters if necessary to restrict the number of assets appearing in the asset list.
  2. Select the assets to be edited. Use ctrl-click to highlight multiple separate assets, or shift-click to highlight a sequential group of assets.
  3. Right-click to open the menu and select Export, or click the Export button from the toolbar. Select the desired Export Style. Generally, Advanced Export is required, so it is selected by default. Simple Export files are suitable for printing.
    1. Export asset references by name if you prefer alphabetical sorting.
  4. Select the Attributes to Export. If you have applied a filter to the list by using an Asset Link or the filter fields (Location, Status, etc.) above the assets list, those filters are applied. If you have selected a View and select Export asset attributes in view, only the attributes visible in that view are exported for the exported assets.

    5. To export all attributes of all exported assets, clear all filters and Export all asset attributes.

  5. Include or exclude binary data. Check to include maps or other image files in your export. Select desired file type to export. Click Export JSON or Export CSV. CSV files can be manipulated with a spreadsheet program such as Microsoft Excel.

Add To or Remove Groups From Selected Assets or Listed Assets

  1. Navigate to User Console > Assets and select an asset link. Use the filters if necessary to restrict the number of assets appearing in the asset list.
  2. Select the assets to be edited. Use ctrl-click to highlight multiple separate assets, or shift-click to highlight a sequential group of assets.
  3. Right-click to open the menu and select Add Group(s) or Remove Group(s). Choose Selected to perform this action on highlighted assets or All to perform this action on all assets in the current list.
  4. In the pop-up box that appears, use the arrows to move groups from Available to Selected. Click OK when finished.

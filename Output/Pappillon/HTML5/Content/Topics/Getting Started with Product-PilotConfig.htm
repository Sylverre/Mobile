Getting Started with CenterScape: Pilot Deployment

This guide provides an overview of the steps necessary to complete a single-site pilot deployment of CenterScape. A pilot deployment uses a local Zone Manager, which is installed with CenterScape, and the included version of the Microsoft SQL Server Express database.

After a successful pilot, a full roll-out can be accomplished using the same basic procedures with a fully enabled database and additional Zone Managers as required. Information about roll-out requirements is included in the Roll-Out section of this guide and in the Systems Administrator manual available at support.rfcode.com

The installation requirements for CenterScape are the same whether deploying as a pilot or a full roll-out to a production environment.

Installation Requirements

Minimum Hardware

  • Dual Core or better processor
  • 2-4 GB RAM
  • 3.4 GB Local Storage

Operating Systems Supported

  • Windows: Microsoft Windows 64-bit Versions
  • Linux: RedHat or CentOS 64-bit Versions

Web Browsers Supported

  • Microsoft Internet Explorer 10 & 11
  • Mozilla Firefox 36 & 37
  • Google Chrome 41 & 42
  • Apple Safari 8

Supported Firmware Versions

  • Version 1.4.1 for the M250 Reader

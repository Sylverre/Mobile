Getting Started with CenterScape: System Configuration

This guide provides an overview of the installation and configuration steps necessary for a deployment of CenterScape

Basic Installation Requirements

Minimum Hardware

  • Dual Core or better processor
  • 2-4 GB RAM
  • 3.4 GB Local Storage

Operating Systems Supported

  • Windows: Microsoft Windows 64-bit Versions
  • Linux: RedHat or CentOS 64-bit Versions

Databases Supported

  • PostgreSQL version 8.3 and above
  • Microsoft SQL Server 2008 and SQL Server 2012

Web Browsers Supported

  • Microsoft Internet Explorer 10 & 11
  • Mozilla Firefox 36 & 37
  • Google Chrome 41 & 42
  • Apple Safari 8

Supported Firmware Versions

  • Version 1.4.1 for the M250 Reader

For the most current release and requirement information, refer to the system requirements KB article in the RF Code Support website: Support.RFCode.com

  1. In your Database Management Application, create a New Database and create or authorize a User with full access to the new database.

  2. Install CenterScape. Do not select the “Install SQL Express” checkbox.

  3. Associate Database with CenterScape

    1. Click the CenterScape icon after the installation is complete and then log in as admin/admin.

    2. Select the database type.

    3. Enter the correct hostname, database name, user ID and password.

    4. Click Test.

    5. Click OK. CenterScape will restart. After five to seven minutes, the database will be populated with the minimal amount of data and the system will be up and running.

  4. Configure Readers
    1. Verify that Readers have been configured using RCU prior to mounting.
    2. Navigate to Configuration > Readers.
    3. Click New and select the reader type from the drop-down list.
      • Name: create a name for the reader
      • Hostname: enter the IP address of the reader
      • UserID/password: if you have enabled user access on the reader, enter the user ID and password
    4. Save Changes.

    If configuring an Up Connection Reader, you must specify the reader Up Connection ID that was assigned to that reader and also if specified, the optional password from the Reader Configuration Utility. For more information, see The Reader Configuration Utility User Manual available at support.rfcode.com

  5. Configure IR Locators [similar to Readers above]
  6. Configure Tag Groups

    A Reader can receive information from up to eight different groups (types) of tags. You will need to configure a tag group for each type of tag you have deployed.

    1. 1. Navigate to Configuration > Tag Groups

    2. Click New and select a treatment code. Treatment codes are printed on the tag labels.

    3. Enter a name for the tag group.

    4. Save Changes.

  7. Add Licenses

    1. Navigate to Configuration > License Keys

    2. Click the Add License Key button and add your 16-digit license key code(s) from the CenterScape License Card

    The System Configuration pane indicates that the System Configuration has been completed.

  8. Add Location Hierarchy
  9. Add Assets to the System

    1. Navigate to User Console > Assets > Manage Assets

    2. Click New and select an asset type from the drop-down list

    3. Type a name for the asset and complete the tag field and description if desired.

    in a pilot deployment, the location drop-down list may not be populated. In that case, leave the field blank. Do not type in a location name.

    1. Click Save.

At this point you have completed the required configuration steps to begin using CenterScape.

For more information on using the CenterScape software please refer to the CenterScape Administrator Manual and CenterScape User Manual that are provided on the CenterScape software CD or available online.

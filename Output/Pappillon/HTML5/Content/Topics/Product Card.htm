CenterScape

CenterScape combines hardware and software to create a real-time environmental monitoring and asset management solution for your data center.

To get started:

  1. Install driver and configuration files to readers and IR locators.
  2. Mount readers and locators in the data center.
  3. Install CenterScape software and a database. (Follow-up:Input license keys, set up tag groups, point CS at reader.)
  4. Affix tags to assets.
  5. Upload locations and inventory to CenterScape.

CenterScape is designed for data center operations and customizable for your particular needs.

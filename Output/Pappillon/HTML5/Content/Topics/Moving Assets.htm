7. On return to Summary Asset tab, new options have appeared: Unlink Summary Asset, Unlink and Move Summary Asset, Delete Summary Asset

  • Use Unlink to remove the association between the location and the asset(s), when you no longer need to monitor that location.
  • Use Unlink and Move to remove the association between the location and the asset(s) when you are moving assets between monitored locations, for instance when moving a rack from one production area to another.
  • Note that IR Rule will not auto-delete, may need to delete or reset.
  • SSI rules retained?

  • Unlinking, which disassociates the asset from a location, will not delete or retire the asset; it will only remove the association of the asset to the location. To actually delete an associated asset you must first perform the disassociation described here.

    Summary assets (locations!) can be created from Admin or User console.

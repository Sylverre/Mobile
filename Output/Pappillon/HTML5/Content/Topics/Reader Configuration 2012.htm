Configuring Readers in CenterScape Readers

Reader Configuration through the Reader Configuration Utility includes IP address assignment and optional security settings, and must be completed before readers are physically deployed. Configuring Readers in CenterScape defines the equipment and channels from which CenterScape will accept data. Both reader configuration steps are required.

Configuring Readers in CenterScape can be done manually through the web interface, or a reader configuration file can be imported.

To configure a reader in the CenterScape web interface:

 

  1. Navigate to Admin Console > Configuration > Readers.
  2. In the Readers configuration screen, click the New button and select the reader type from the drop-down list.
  1. Set the Basic and Network configuration settings for the reader:
  • Reader: Choose the type of reader you wish to configure from the drop-down list.
  • Name: Create a name for the reader.
  • Zone Manager: Choose the Zone Manager that you would like to assign your reader to (by default this is your local Zone Manager).
  • Description: Enter a description for this Reader.
  • Enabled: If you would like your reader to go active and receive and transmit tag data after saving your configuration, check this box.
  • Hostname: Enter the IP address of your reader in this field.
  • Port: Enter the port number over which to communicate with your reader (by default this is 6500).
  • SSL Mode: Select the desired SSL mode:
    • OFF: turns SSL mode off on the reader
    • IFAVAIL: uses SSL on the reader if available
    • REQUIRED: requires use of SSL
    • STRICT: will authenticate the matching hostname and only connect if the hostname matches the reader

Accept the default entries where possible. If you are planning to use multiple Zone Managers, it may be simpler to complete the initial configuration using the single Local Zone Manager and then changing reader associations once the other Zone Managers have been installed. Refer to the Advanced Reader Configuration page of this document for more information.

  1. Click the Save Changes button.
    The reader you have just configured appears in the middle pane to the left of the configuration pane.

If configuring an Up Connection Reader, you must specify the reader Up Connection ID that was assigned to that reader and also, if specified, the optional password from the Reader Configuration Utility. Please refer to the Reader Configuration Utility User Manual for more information.

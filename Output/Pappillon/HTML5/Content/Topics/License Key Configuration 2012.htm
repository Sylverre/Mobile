License Keys

A single license key must be configured after CenterScape is installed. Additional asset or advanced feature licenses may be purchased at any time.

  1. Navigate to Configuration > License Keys.
  2. Click the Add License Key button and enter the 16-digit license key code(s) provided by RF Code. Click OK when finished.
  3. The license key appears in the list with the license key parameters of License Count, Expiration Date, and License Key Type.

Default Dashboards

CenterScape includes a set of standard dashboards to help you monitor and manage your data center:

  • Current Inventory Dashboard: Default is All Locations, but easy to copy/modify for individual locations
  • Door Monitoring Dashboard: Current open/closed and chart for recent open/closed trends
  • General Environmentals Dashboard: Shows current values and recent trends
  • Environmental Thresholds Dashboard: Compares current values to configured min/max values
  • Storage Dashboard: Shows assets in storage now and chart counting assets that have been in storage 30, 60, 90, and 120+ days. Does not show decommissioning/decommissioned assets
  • Temperature Dashboard: Shows current values and recent trends
  • Weight Capacity Dashboard: Shows current min/max/ave overall weight capacity and lists each location (rack) whose weight is nearing or exceeding capacity.
  • Shows chart comparing current ownership totals
  • Depreciation Dashboard: Shows aggregated purchase values and depreciated values for assets in locations
  • Operational Status Dashboard: Chart showing asset totals grouped by operational status. Also, chart showing recent trends in operational status.
  • Inventory Warranty Dashboard: Displays totals of assets under warranty and how much warranty is left
  • Location Audit Dashboard: Shows line chart showing trend and grid showing NIEL assets
  • Scheduled Maintenance Dashboard: Shows totals/percents of assets which are up-to-date on their maintenance. Also lists assets who are approaching or have exceeded their required maintenance date.
  • CRAC Unit Dashboard: Shows CRAC Unit supply, return, and RTI values. Also has chart showing recent trends
  • Power Monitoring Dashboard(s):
  • U-Space Capacity Dashboard: Shows total U-space capacity compared to available U-space and recent trends

Standard dashboards can be modified by the systems administrator. Custom dashboards can also be created.

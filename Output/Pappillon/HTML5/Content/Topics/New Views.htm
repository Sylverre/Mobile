Views

Views define what data is presented to users and how information is displayed.  CenterScape includes a number of standard views. The System Administrator can grant or restrict access to these Views and can edit Views or create new Views at any time.

 

Until or unless a standard view is edited by a System Administrator, the attributes for a view are as listed here:

  • Data Center View: Shows total equipment count, non-operational equipment count, total U space, available U space, total rack count and operational rack count.
  • Default View: Displays asset alerts, asset operational status, tag IDs, summary location assignment, and location verification status.
  • Equipment View:Shows asset type, manufacturer, model, serial number, mount type, and operational status.
  • Location Verified View: Displays assigned location, assigned to (summary) location, location verified status, operational status, last operational change, and asset tag.
  • Rack Capacity View: Shows rack assigned location, equipment count, rack U space capacity, available U space, U space utilized, contiguous available U space, rack configuration status, current rack power and power capacity consumed.
  • Rack RCI View: Displays Rack Cooling Indexes RCI(HI) and RCI(LO), Max Intake Temperature and Humidity, Max Exhaust Temperature, and Average Temperature Delta.
  • Rack Status View: Shows rack operational status, all assets operational status, rack configuration status, assigned location, equipment count, and all assets verified status.
  • Single Temp Sensor View: Shows the current online status, temperature, humidity, dew point, airflow position, rack position, and asset tag ID of a sensor.
  • System View: Displays each assetâ€™s name, manufacturer, model, and rack item U height.
  • Update View: Shows the last update user and time, assigned location, and detected location.


Views for which a User has been granted access, either individually or through Group membership, appear as options in the drop-down menu beneath the RF Code logo on the user console.

 

To Create a New View

  1. Navigate to Admin Console > Customization > Views and click New.
  2. Complete all required and any desired additional fields. NOTE:  Those fields with asterisks (*) are required fields.
  3. Click Save Changes.

To Edit a View

  1. Navigate to Admin Console > Customization > Views. Select the View to edit and update the fields.
  2. Click Save Changes.

Import Location Hierarchy Using Spreadsheet

 

  • File must be created in and exported from CenterScape, then populated and imported into system.
  • Every Location must have a unique name.

As a best practice, structure the levels of your Location Hierarchy to fit within CenterScape's data schema. The schema's expected levels are:

  • Country
  • State
  • City
  • Campus
  • Building
  • Floor
  • Zone
  • Room

Use as many of these levels as are appropriate for current or anticipated needs.

  1. Though it is not required, it is recommended that you create one or two locations manually before exporting a location configuration.
  2. Navigate to Admin Console > Export Configuration. Deselect all variables except for Locations & Rules. Select CSV and click Export Configuration.

 

  1. Open file in spreadsheet program.
  2. Add Locations. Each location must have a unique name and a parent. All other fields are optional.
  3. Save file.
  4. Browse to and Upload the system configuration file.

 

The location hierarchy is part of the system configuration. Use the Zone Manager configuration (bottom) task pane only to import Zone Manager configuration files.

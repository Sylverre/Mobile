Default Alerts

  • Door Off Hours Alert: Alerts when doors are opened during times when they shouldn't be
  • Door Open Duration Alert: Alerts when a door stays open longer than the specified max duration
  • Fluid Detected Alert
  • Temperature Too High Alert: Alerts when temperature for a location exceeds configured max
  • Temperature Too Low Alert: Alerts when temperature for a location is below configured min
  • Humidity Too High Alert: Alerts when humidity for a location exceeds configured max
  • Humidity Too Low Alert: Alerts when humidity for a location is below configured min
  • Approaching Dew Point Alert: Alerts when dew point delta is below configured min
  • Precipitation Alert: Alerts when dew point delta is at or below zero
  • Asset Storage Threshold Exceeded Alert: Alerts when an asset is currently in storage and whose total days in storage exceeds the configured max. Clears when the asset is moved out of storage or if operational status shows that it's being decommissioned
  • Weight Exceeded Alert: Alerts when weight exceeds configured maximum
  • Approaching Planned Replacement Alert: Alerts when an asset is still in service but is approaching its planned replacement date
  • Planned Replacement Exceeded Alert: Alerts when an asset is still in service but has exceeded its planned replacement date
  • Critical Warranty About To Expire Alert: Alerts when an asset which is required to always have a warranty is approaching the end of its warranty
  • Critical Warranty Expired Alert: Alerts when an asset which is required to always have a warranty actually has an expired warranty
  • Not In Expected Location Threshold
  • Scheduled Maintenance Not Completed Alert
  • Approaching SLA Threshold: Alerts when a value is approaching an SLA threshold
  • SLA Threshold Exceeded: Alerts when a value has exceeded an SLA threshold
  • Containment Alert: Alerts when a containment has an environmental value that exceeds the configured min/max value
  • CRAC Environmental Alerts: Alerts when temp/humidity/dew-point values exceed thresholds
  • Power Monitoring Alerts Threshold(s)

Roll-Out

If any issues are discovered during the pilot deployment, examine plans and make adjustments before beginning the full roll-out.

For a full roll-out, you need to:

  1. Create a database
  2. Install additional Zone Managers if required for your Deployment Configuration
  3. Affix tags*
  4. Mount Readers*
  5. Mount IR Locators*
  6. Configure CenterScape
  7. Populate CenterScape
  8. Add Users
  9. Customize CenterScape

*Asterisks indicate physical Deployments. The remaining items may be completed by the Systems Administrator at a different time if so desired.

Details about software configuration and system administration are covered in the System Administration Manual available at support.rfcode.com

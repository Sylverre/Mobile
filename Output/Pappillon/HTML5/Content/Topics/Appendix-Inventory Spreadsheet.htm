Inventory Spreadsheet

If you completed an inventory during your site assessment (pre-purchase) process, use that inventory as a data source for importing into CenterScape. Otherwise, divide your site into zones and list all assets within each zone in a spreadsheet. For each asset, record information to be included in CenterScape.

What information you collect depends on your needs. Commonly collected information includes:

  • Asset Name
  • Asset Type
  • Asset Description
  • Manufacturer
  • Model
  • Serial Number
  • Mount Type
  • Purchase Date
  • Location, current

A sample inventory spreadsheet can be downloaded from support.rfcode.com

To import assets, you must first export a CSV file, then populate it (fill in its cells) and import back to the system. The CSV file will have seven columns, but only two of these must be completed: the asset ºs Name ($aName) and the assigned Asset Tag ($aAssetTag). Both of these must be unique; duplicates will not be imported. Other columns may be completed if the information is available.

Create a Map

An image must be uploaded to CenterScape for each map created. Floor plans and schematics make ideal maps. To create a map in CenterScape, perform the following steps:

  1. In the Admin Console, navigate to Locations/Rules/Maps > Map Configuration. 
  2. From the Location hierarchy in the middle pane, select the level on which you want the map to be displayed.
  3. Click New to create a new map.
  4. Complete all required and any desired additional fields. NOTE:  Those fields with asterisks (*) are required fields.


  • Map Family: Create categories for your maps. This field is empty at first; however, every category you create is added to the drop-down list for new maps. Common families include Schematic, Data Center, Floor Plan.
  • Name: Assign a name to the map that lets the user know what the map represents.
  • Default Map View: Select the view with which this map opens.  
  • Is Default Map: Check this box to set this map as the default map for this location.  
  • Map Views: Select the desired Map Views to apply for this Map. Map Views define which sensors, attributes, and/or data can be displayed for the respective Hot Spots in the map.  This option can restrict specific views of the map to certain users. 

To add or edit a Map View, click Map Views in the Locations/Rules/Maps task list.

  • Map Image: Upload a graphic to serve as the background of the map. Click the Ellipsis [...] button to browse to the desired file.
  • Hot Spot Display Settings: If desired, select Hot Spot display settings for this map. Settings here override the Global Settings.

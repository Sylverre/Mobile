Tag Management

Tags must be detected, assigned, and associated with assets.

Need best-practice description for deployment.

  1. Switch to the User Console > Tag Management > Manage Tags.
  1. To move tags to the Unassigned Tags list, select a Tag Group from the drop-down menu.

NOTE: The drop-down menu defaults to All, which if left unchanged allows you to import all of the tags in all of your Tag Groups at once.

  1. Click Add All Tags.
  1. All of the detected tags that were in the list in the middle pane will now appear in the right pane in the list of Unassigned Tags.

NOTE: After the pilot, at the time of your full roll-out, you will have the option to use the export and import functions to associate tags with your Assets in bulk outside of the system for speed and ease; this process is covered later in this document.

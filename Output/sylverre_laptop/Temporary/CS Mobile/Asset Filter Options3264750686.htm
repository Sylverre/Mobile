Asset Filter Options

Type Filters

Asset (NAV) – This is the type of asset that you want to filter for. This variable is a function of, and is controlled by, what screen you are on in the application. To change the value, you must first navigate to the screen that you want to perform the filter on, and then select the Filter button.

If you were not sure where you are in the CenterScape Mobile application when you want to set up a Filter, you can get to the to get to the top three levels by selecting the Main Menu button and choosing one of the top levels listed here:

  • Browse Assets by Type
  • Browse Assets by Location
  • Browse Alerts
